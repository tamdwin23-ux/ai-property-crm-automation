# 🏠 AI Property CRM & Sales Automation

> An AI-powered property management and CRM automation system built with **n8n, OpenAI, Supabase, PostgreSQL, Gmail, and Google Calendar** to automate the complete real estate sales process from lead capture to property viewing.

---

# 🚀 Business Problem

Real estate agents spend a significant amount of time:

- Responding to repetitive enquiries
- Qualifying potential buyers
- Matching customers with suitable properties
- Scheduling property viewings
- Updating CRM records
- Following up with clients
- Sending appointment confirmations

These repetitive tasks reduce productivity and increase response times.

---

# 💡 Solution

This AI-powered workflow automates the entire customer journey by:

- Capturing property enquiries
- Identifying customer information
- Qualifying leads using AI
- Recommending matching properties
- Assigning the appropriate sales agent
- Scheduling property viewings
- Creating Google Calendar events
- Sending confirmation emails
- Updating the CRM automatically
